Glossário em Libras

Termos do glossário

Lista de conceitos usados neste projeto acadêmico, com suporte visual em Língua Brasileira de Sinais (Libras).

Acessibilidade

Sistema acessível. Todas as pessoas podem usar.

Login

Entrar no sistema usando e-mail e senha.

Cadastro

Criar conta no sistema.

Usuário

Pessoa que usa o sistema.

Evento

Atividade disponível no sistema.

Inscrição

Participar do evento. Fazer registro.

Certificado

Documento que comprova participação.

Frontend

Parte visual do sistema. Tela que o usuário vê.

Backend

Parte interna do sistema. Processa dados.

API

Comunicação entre frontend e backend.

Banco de dados

Local onde as informações ficam armazenadas.

Interface

Tela do sistema. Onde o usuário interage.

Logout

Sair do sistema.
